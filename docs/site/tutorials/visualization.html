Visualizing Audio Features

Spectrograms, waveforms, and chromagrams. If Mom can see it, Mom can fix it.

Setup

Install MetalMom with the display extra to pull in matplotlib.

Terminal 
pip install metalmom[display]
Python 
import metalmom
import matplotlib.pyplot as plt
import numpy as np

Waveform Display

The simplest visualization: the raw amplitude over time. Good for checking that your file loaded correctly and spotting obvious issues like clipping or silence.

Python 
y, sr = metalmom.load("song.mp3")

fig, ax = plt.subplots(figsize=(14, 3))
metalmom.display.waveshow(y, sr=sr, ax=ax)
ax.set_title("Waveform")
plt.tight_layout()
plt.show()

Mel Spectrogram

The workhorse of audio analysis. Frequency content over time, warped to match human hearing. This is what most ML models see.

Python 
S = metalmom.melspectrogram(y=y, sr=sr, n_mels=128)
S_db = metalmom.amplitude_to_db(S, ref=1.0)

fig, ax = plt.subplots(figsize=(14, 4))
img = metalmom.display.specshow(S_db, sr=sr, x_axis='time', y_axis='mel', ax=ax)
ax.set_title("Mel Spectrogram")
plt.colorbar(img, ax=ax, format='%+2.0f dB')
plt.tight_layout()
plt.show()

Chromagram

Pitch class energy over time. Twelve bins (C through B), perfect for visualizing harmonic content, chord changes, and key.

Python 
chroma = metalmom.chroma_cqt(y=y, sr=sr)

fig, ax = plt.subplots(figsize=(14, 4))
img = metalmom.display.specshow(chroma, sr=sr, x_axis='time', y_axis='chroma', ax=ax)
ax.set_title("Chromagram (CQT)")
plt.colorbar(img, ax=ax)
plt.tight_layout()
plt.show()

MFCCs

Mel-frequency cepstral coefficients — a compact representation of spectral shape. Widely used for speech and music classification.

Python 
mfcc = metalmom.mfcc(y=y, sr=sr, n_mfcc=13)

fig, ax = plt.subplots(figsize=(14, 4))
img = metalmom.display.specshow(mfcc, sr=sr, x_axis='time', ax=ax)
ax.set_title("MFCCs")
plt.colorbar(img, ax=ax)
plt.tight_layout()
plt.show()

Beats on Waveform

Combine features: overlay detected beat positions on the waveform. A quick sanity check that your beat tracker is finding the right spots.

Python 
tempo, beat_frames = metalmom.beat_track(y=y, sr=sr)
beat_times = metalmom.frames_to_time(beat_frames, sr=sr)

fig, ax = plt.subplots(figsize=(14, 3))
metalmom.display.waveshow(y, sr=sr, ax=ax, alpha=0.6)
for bt in beat_times:
    ax.axvline(x=bt, color='#f59e0b', alpha=0.6, linewidth=0.8)
ax.set_title(f"Waveform with beats ({tempo:.0f} BPM)")
plt.tight_layout()
plt.show()

Multi-panel Dashboard

Put it all together: four views of your audio in a single figure. Waveform, mel spectrogram, chromagram, and MFCCs, all time-aligned.

Python 
fig, axes = plt.subplots(4, 1, figsize=(14, 12), sharex=True)

# Waveform
metalmom.display.waveshow(y, sr=sr, ax=axes[0], alpha=0.6)
axes[0].set_title("Waveform")

# Mel spectrogram
S_db = metalmom.amplitude_to_db(metalmom.melspectrogram(y=y, sr=sr))
metalmom.display.specshow(S_db, sr=sr, x_axis='time', y_axis='mel', ax=axes[1])
axes[1].set_title("Mel Spectrogram")

# Chromagram
metalmom.display.specshow(metalmom.chroma_cqt(y=y, sr=sr), sr=sr, x_axis='time', y_axis='chroma', ax=axes[2])
axes[2].set_title("Chromagram")

# MFCCs
metalmom.display.specshow(metalmom.mfcc(y=y, sr=sr), sr=sr, x_axis='time', ax=axes[3])
axes[3].set_title("MFCCs")

plt.tight_layout()
plt.show()

Four views of your audio. Mom sees everything.

Customization Tips

The specshow function accepts the same parameters as librosa.display.specshow. Here are the most useful ones.

Parameter Description Example
x_axis Time axis format 'time', 'frames', 's'
y_axis Frequency axis 'mel', 'hz', 'log', 'chroma'
cmap Colormap 'magma', 'viridis', 'inferno'
vmin/vmax Value range vmin=-80, vmax=0 for dB
For dB-scale spectrograms, setting vmin=-80 and vmax=0 gives consistent visual contrast regardless of signal level.
