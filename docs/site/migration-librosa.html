Coming from librosa?

Good news: you already know MetalMom. She just runs faster.

The Two-Line Migration

Option 1 — Swap the import 
# Before
import librosa

# After
import metalmom as librosa
Option 2 — Use the compat shim 
from metalmom.compat import librosa

That's it. Your existing code works. Mom's a drop-in.

API Differences

Almost everything is identical, but Mom has a few opinions:

Topic librosa MetalMom
Namespace librosa.feature.melspectrogram() metalmom.melspectrogram() (flat)
Compat namespace — metalmom.compat.librosa.feature.melspectrogram()
Platform Linux, macOS, Windows macOS only (that's where the Metal is)
Float precision float32 or float64 float32 always
Audio backend soundfile / audioread AVFoundation (more formats)
GPU acceleration None Automatic via Smart Dispatch
The compat shim gives you the exact librosa namespace structure if your code needs it. But most folks prefer Mom's flat API — less typing, same results.
