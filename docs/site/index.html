Your librosa code already works

Swap the import. Keep everything else.

librosa 
import librosa

y, sr = librosa.load("song.mp3")
S = librosa.feature.melspectrogram(y=y, sr=sr)
tempo, beats = librosa.beat.beat_track(y=y, sr=sr)
MetalMom 
import metalmom

y, sr = metalmom.load("song.mp3")
S = metalmom.melspectrogram(y=y, sr=sr)
tempo, beats = metalmom.beat_track(y=y, sr=sr)

Or just alias it: import metalmom as librosa — Mom won't tell.
