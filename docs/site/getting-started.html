Getting Started

Mom's getting you set up. It'll only take a minute.

Requirements

  • macOS 13 (Ventura) or later
  • Apple Silicon (M1, M2, M3, M4) or Intel Mac with compatible GPU
  • Python 3.11 or later
MetalMom is macOS-only. That's where the Metal is.

Installation

One line. Mom keeps it simple.

Terminal 
pip install metalmom

Optional extras

Terminal 
# Visualization support
pip install metalmom[display]    # + matplotlib

# Evaluation metrics
pip install metalmom[eval]       # + mir_eval

# Development (testing + librosa parity)
pip install metalmom[dev]        # + pytest, librosa

Your First Analysis

Load a file, compute a spectrogram, detect some beats. The whole pipeline in ten lines.

Python 
import metalmom

# Load an audio file
y, sr = metalmom.load("song.mp3")
print(f"Loaded {len(y)/sr:.1f} seconds at {sr} Hz")

# Compute a mel spectrogram
S = metalmom.melspectrogram(y=y, sr=sr)
S_db = metalmom.amplitude_to_db(S, ref=1.0)
print(f"Spectrogram shape: {S_db.shape}")

# Detect beats
tempo, beat_frames = metalmom.beat_track(y=y, sr=sr)
print(f"Tempo: {tempo:.1f} BPM, {len(beat_frames)} beats detected")

That's it. If you're coming from librosa, you'll notice the API is identical. Mom raised it that way.

GPU Acceleration

MetalMom's Smart Dispatch automatically routes operations to the fastest backend. No configuration needed — Mom handles it.

Operation Backend Threshold
STFT Metal > 8192 samples
Element-wise ops Metal > 65536 elements
Matrix multiply Metal > 512×512
Reductions Metal > 131072 elements
Convolution Metal > 32768 elements
MFCC pipeline Metal (fused) > 8192 samples
Everything else Accelerate/vDSP Always

Below these thresholds, Accelerate is actually faster due to Metal dispatch overhead. Mom knows the crossover points.
